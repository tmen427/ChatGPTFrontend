Misc.

Information about this app

  • The frontend of this app is written in Angular.js with Bootstrap and the backend is written in Node.js
  • LangChain.js was used in order to help upload our own personal documents with our language model which in this case is OpenAi ChatGPT
  • LangChain.js supports various file types such as .CSV and .TXT, but bigger files will cause longer response times
  • The site is currently being hosted on an Amazon EC2 instance with Nginx being used as the reverse proxy to the Node.js backend
